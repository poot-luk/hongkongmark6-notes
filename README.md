# hongkongmark6-notes

## 项目简介

`hongkongmark6-notes` 是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定网站或域名，仅作为通用 HTML 页面资料的存储与展示空间。页面内容涵盖笔记、工具、参考信息等，以静态 HTML 形式提供。

## 目录说明

```
.
├── index.html          # 仓库入口页面（可选）
├── pages/              # 独立 HTML 页面目录
│   ├── page1.html
│   ├── page2.html
│   └── ...
├── assets/             # 静态资源（图片、样式、脚本等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/`：存放各个独立 HTML 页面，每个页面自包含内容与样式。
- `assets/`：存放页面引用的公共资源，如图片、样式表、JavaScript 文件等。

## 页面归档说明

本仓库中的 HTML 页面均为独立归档，可直接通过浏览器打开查看。页面内容可能包含笔记、数据整理、工具使用说明等，不保证时效性或完整性。页面之间无依赖关系，可单独使用。

## 维护与更新

- 本仓库不定期更新，新增或修改页面会通过 Git 提交记录体现。
- 欢迎通过 Issue 或 Pull Request 提交建议或修正，但请注意本仓库不针对特定网站或服务，不接受与具体域名、营销推广相关的内容。
- 页面内容仅供个人学习与参考，不构成任何形式的指导或承诺。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接双击打开 `pages/` 目录下的任意 HTML 文件，或通过本地 HTTP 服务器访问。
3. 也可通过 GitHub Pages 等功能在线浏览（需自行配置）。
